# ResQ Link 
**Disaster Management & Emergency Response Web System**

ResQ Link is a lightweight web-based disaster management system designed to help during emergency situations by providing quick access to alerts, resources, and essential information.  
The project is built using **HTML, CSS, and JavaScript**, focusing on simplicity, speed, and accessibility.

---

## Features

- Emergency alerts & disaster information
- Location-based awareness (manual or browser-based)
- Quick access to emergency contacts
- Resource and safety guidelines during disasters
- Simple, responsive UI for fast access on any device
- No backend dependency (runs directly in browser)

---

## Tech Stack

- **HTML5** – Structure of the application    
- **JavaScript** – Core logic and interactivity

## How to Run Locally

- Clone the repository
   ```
   git clone https://github.com/your-username/res-q-link.git
   ```
- Navigate to the project folder
  ```
  cd res-q-link
  ```
- Open index.html in your browser (No server setup required)
